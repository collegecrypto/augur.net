Prototype

We're currently alpha-testing the Augur prototype. Our installation procedures and other documentation will get better/easier as time goes on and we smooth out the rough edges! In the meantime, please email us or join us in #augur on FreeNode IRC if you have trouble installing or running the prototype.

Installation

Windows

Download and run the Windows installer: Augur_0.1.1.exe

After running the installer, click on your Start Menu, navigate to the Augur application folder, then click on Augur. Then open your web browser and enter http://localhost:9000 in the address bar, and you should be up and running!

Note: so far, this installer has been tested on Windows 7.1 and 8.1 (both 64-bit). If the installer doesn't work for you, click here for manual install instructions.

Manual installation

  1. First, make sure you have Python 2.7 installed.
  2. You need to install a few dependencies, namely numpy, six, cdecimal and pip, which can all be found here. I recommend using Ctrl+F to find them (and make sure you get the newest one that matches your python installation's architecture!)
  3. Once those are installed, you need to download ez_setup.py (right click on the link and click "Save Target As" and save with the name ez_setup.py.) You should be able to double click this file if you have installed python, and it will run and close on it's own.
  4. Now you need to edit the Path environment variable. To do this, go to the Start menu, right click on Computer, click on Advanced System Settings in the upper left, click Environment Variables in the bottom right, then in the System Variables section, scroll down till you see Path. Click on Path, and click Edit.... Press the Home key to get to the start of the line, then add C:\Python27\Scripts;C:\Python27\; to the start of the line and click OK.
  5. Now go to http://aka.ms/vcpython27 to download and install the Windows C compiler for use with pip.
  6. Finally, go back to the Start Menu, type cmd and press enter, then type pip install flask flask-socketio.

Run the Augur client by going into the augur directory and double-clicking on augur.py.

MacOS X

Download the installer, navigate to your download folder, and double-click on Augur.command.zip. Then open your web browser and enter http://localhost:9000 in the address bar.

Install script not working? Click here for manual install instructions!

Manual installation

To install the prototype from source on MacOS X, you must have the following software installed:

  • Python 2.7
  • pip - just type sudo easy_install pip in the terminal to install!
  • git
  • Xcode command line tools

Open a terminal and enter the following commands:


  git clone https://github.com/AugurProject/augur.git
  cd augur
  sudo pip install -r requirements.txt

To run Augur, type:


  ./augur_ctl start

Linux

First, install the necessary dependencies:


  apt-get install git python-numpy libpython-dev python-leveldb
  pip2 install m3-cdecimal

Next, download the Augur client:


  git clone https://github.com/AugurProject/augur.git
  cd augur
  pip install -r requirements.txt

Then, to run augur, type:


  ./augur_ctl start

Note: Augur has only been tested so far in Ubuntu 12.04 and 14.04. If you've tested this on another Linux distro, let us know!
