Prototype

Installation

To install the prototype, you must have the following software installed:

  • Python 2.7
  • Truthcoin-POW
  • git
  • pip

OSX

Install Xcode command line tools:


  $ git clone git@github.com:scottzer0/Augur-UI.git
  $ cd Augur-UI
  $ sudo pip install -r requirements.txt
  $ python app.py

Linux

First, checkout Augur-UI next to Truthcoin-POW (or change config path in app.py). Then enter the following commands:


  $ cd Augur-UI
  $ pip install -r requirements.txt
  $ python app.py

Windows

Unfortunately, Windows installation requires a few more steps than the various *nix platforms.

  1. First, make sure you have Python 2.7 installed. Also, make sure that you have downloaded the prototype core. Both programs will need to be extracted to the same directory if you download the zip files.
  2. You need to install a few dependencies, namely numpy, six, cdecimal and pip, which can all be found here. I recommend using Ctrl+F to find them (and make sure you get the newe$
  3. Once those are installed, you need to download ez_setup.py (right click on the link and click "Save Target As" and save with the name ez_setup.py.) You sh$
  4. Now you need to edit the Path environment variable. To do this, go to the Start menu, right click on Computer, click on Advanced System Settings in the upper left, click Environment Variables in the bottom right$
  5. Now go to http://aka.ms/vcpython27 to download and install the windows C compiler for use with pip.
  6. Finally, go back to the Start Menu type cmd and press Enter, then type pip install flask flask-socketio.

You can run the UI by double clicking on app.py.

Caution: the Augur prototype is built for rapid testing. Our team makes no guarantees about its security or scalability! Use at your own risk :)
