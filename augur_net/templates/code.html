Source Code

The Augur project is split into several Github repositories:

  • Augur

    • URL: https://github.com/AugurProject/augur
    • Maintainers: Scott Leonard & Jack Peterson
    • Augur runs as a local application, and is accessed via web browser. This package is intended to be used with both the prototype and production versions of Augur's modular core, which handles all the low-level networking and blockchain operations.

  • Prototype Core

    • URL: https://github.com/zack-bitcoin/augur-core
    • Maintainer: Zack Hess
    • Zack's Python implementation of the Augur/Truthcoin core, which extends Basiccoin's source code. Contains all the basic networking, mining, cryptography, event/market creation, and consensus algorithms required for decentralized prediction markets. Includes CLI commands similar to Bitcoin's RPC command suite.

  • Website

    • URL: https://github.com/AugurProject/augur.net
    • Maintainers: Jack Peterson & Scott Leonard
    • Source code for this website. This may be just a boring, static website, but Jack and Scott built it using Django anyway because they both think Python is pretty okay.

  • Sidecoin

    • URL: https://github.com/AugurProject/Sidecoin
    • Maintainers: Joey Krug & Jack Peterson
    • Sidecoin is a modified version of Bitcoin Core 0.9.1. It allows Bitcoin holders to trustlessly be awarded coins from a "snapshot" of the blockchain (e.g., awarded Sidecoins in an amount proportional to the number of Bitcoins they hold). Sidecoin was created by Joey and Jack as a solution to the "Spin-off" problem.

  • Camera

    • URL: https://github.com/AugurProject/camera
    • Maintainer: Joey Krug
    • Joey's memory-lean version of znort's C++ blockparser. We used Camera to take a "snapshot" of the Bitcoin blockchain -- we took all the public addresses in the network with a substantial balance (over about 20 cents in USD) and sorted them from highest to lowest balance.

  • Consensus algorithm

    • URL: https://github.com/AugurProject/pyconsensus
    • Maintainer: Jack Peterson
    • Standalone Python version of the Principal Component Analysis-based consensus algorithm used by Augur. This implementation was a joint effort between Jack and Paul Sztorc. It is based on R code from the Truthcoin repo.

  • Truthcoin whitepaper

    • URL: https://github.com/psztorc/Truthcoin
    • Maintainer: Paul Sztorc
    • The theoretical foundation of the Augur project. (It inspired all the rest of this stuff!) This paper covers -- invents, in fact -- the Principal Component Analysis-based consensus algorithm used by our software. It also details Augur's game-theoretic underpinning. This repository includes sample code written in R, Python, and C++.

If you have questions about a particular repo -- or just want to pitch in! -- contact the repo's maintainer, create an issue on Github, or just start hacking away and send us a pull request.
